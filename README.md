## Pomodoro-Timer
## How it works:
+ First button: Toggle (Start and Stop)
+ Second button: Set timer to 25 minutes (Pomodoro)
+ Third button: Set timer to 15 minutes (Break)

## Wiring:
DS1302 RTC
+ 1 VCC
+ 1 GND
+ 3 Digital (D6,D7,D8)


TM1637 display
+ 1 VCC
+ 1 GND
+ 2 Digital (D4,D5)


3 Buttons
+ 1 GND
+ 1 Digital (D2,D3,D10)


Buzzer
+ 1 GND
+ 1 Digital (D9)
